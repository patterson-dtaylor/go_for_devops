### Variables, constants, and data types
- Different ways to call vars
  - Keyword:
    ```go
    var name string
    var age int
    ```
  - Multiple Values
    ```go
    var (
      firstName, lastName string
      age int
    )
    ```
  - Init Values
    ```go
    var name sring = "Taylor"
    var age int = 40
    ```
  - Type Inference
    ```go
    var name = "Taylor"
    var age = 40
    ```
  - Short Declaration
    ```go
    name := "Taylor"
    age := 40
    ```
- Constants
  - Defined like vars, but with a constant keyword
    ```go
    const pi = 3.14
    ```
